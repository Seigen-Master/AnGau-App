# **App Name**: AnGau Light

## Core Features:

- Role-Based Access: Role-Based Access: Implement login with distinct dashboards for admins and caregivers.
- Simple Schedule View: Simplified Caregiver Schedule View: Show a straightforward daily schedule view for caregivers, focusing on shift times, tasks, and patient details without real-time updates.
- Patient Info Display: Patient Info Display: Provide a static display of patient information, including basic health conditions and addresses.
- Messaging Simulation: Messaging Simulation: A simplified messaging feature to simulate communication between caregivers and admins.
- Admin Dashboard: Admin Dashboard: Enable admins to manage caregivers, and view/create schedules
- AI Assisted Planning: Suggested next steps: Use an AI tool to reason about possible next tasks that should be assigned to the care giver to the current patient.

## Style Guidelines:

- Primary color: A soft blue (#72BCD4) to convey trust and calmness, crucial for a healthcare application.
- Background color: A very light, desaturated blue (#F0F8FF) to maintain a clean and professional appearance.
- Accent color: A gentle green (#8BC34A), used sparingly for positive confirmations or highlights.
- Body and headline font: 'PT Sans' (sans-serif) to provide a balance of modern appeal with some warmth, so as to remain approachable and humanistic.
- Use clear and universally recognizable icons for navigation and key actions; follow common convention for clock-in/out, messaging, and user profile.
- Maintain a clean and well-spaced layout to prevent clutter. Focus on readability and intuitive navigation to ensure ease of use for both admins and caregivers.
